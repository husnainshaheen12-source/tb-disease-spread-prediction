# AI-Based Tuberculosis Spread Simulation and TB Case Prediction System

## Project Overview

Tuberculosis, also known as TB, is a bacterial infectious disease that mainly spreads through the air when an infected person coughs, sneezes, or has close contact with other people.

This project uses data analysis, machine learning, and simulation to study TB spread and predict future TB cases.

## Objectives

- Analyze tuberculosis dataset using Python.
- Identify TB trends by country and year.
- Train machine learning models to predict TB cases.
- Create a TB disease spread simulation.
- Build an interactive Streamlit dashboard.
- Present results using graphs, predictions, and simulation output.

## Final Dashboard Features

The final dashboard includes dataset analysis, machine learning prediction, standard simulation, and live visual simulation with policy alerts for masks and lockdown.
