# Final Project Summary

## Project Title

AI-Based Tuberculosis Spread Simulation and TB Case Prediction System

## Summary

This project analyzes tuberculosis data, trains machine learning models, predicts estimated TB cases, and simulates TB disease spread using Python and Streamlit.

- Added Streamlit dashboard.
